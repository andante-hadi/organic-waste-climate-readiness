# GWP20 Pathway Sensitivity Results

Date: 2026-05-23

## Purpose

This sensitivity tests how the deterministic four-pathway screen changes under a 20-year methane climate metric. The main manuscript uses GWP100, while GWP20 is useful for near-term methane mitigation framing.

- `data/processed/country_ofmsw_four_pathway_comparison_gwp20.csv`

## Interpretation

The GWP20 screen strongly increases the value of pathways that avoid disposal methane from already-generated organic waste. Under this near-term methane framing, AD-bio-CNG becomes the leading deterministic pathway in most countries with sufficient data. This does not overturn the waste hierarchy because prevention remains the upstream priority, but it reinforces the paper's central claim that hierarchy alone is insufficient for climate-investment sequencing.

The GWP20 sensitivity should be presented as supplementary evidence. The main text can state that near-term methane weighting strengthens the case for immediate diversion of unmanaged organic waste, especially where prevention is slow to implement or waste has already been generated.

## Caveat

This GWP20 sensitivity reuses central pathway assumptions and changes only methane climate weighting. It does not yet include landfill gas capture, oxidation, methane slip, or nutrient credits.
