# Submission Readiness Memo

## Current Status

The manuscript is now being prepared as a Nature Sustainability **Analysis** rather than a conventional Article. That category fits the study better because the contribution is a new comparative analysis of existing global datasets, pathway parameters, uncertainty and readiness indicators.

The current-data strategy is defensible: submit once using the transparent ecoinvent 3.12 documentation plus ecoinvent 3.1 biogas-yield proxy, with sensitivity and stress tests around the vulnerable parameters. If Nature Sustainability rejects the manuscript mainly because of the ecoinvent limitation, then purchase or obtain newer ecoinvent access before redirecting the paper to another journal.

The core scientific story is coherent:

- Food and green organic municipal waste is a large methane-relevant flow.
- Prevention remains the most robust country-level principle.
- AD-bio-CNG is high-potential but assumption- and readiness-sensitive.
- Composting is conservative in the central climate-only screen and has broader soil/nutrient value.
- Readiness separates deployable mitigation from theoretical mitigation.

## Strongest Elements

1. **Clear Nature Sustainability framing**

The paper is framed as operationalizing the waste hierarchy under climate, uncertainty and readiness constraints, not as a simple technology ranking.

2. **Global scope with policy relevance**

The analysis covers 217 countries/economies and produces country classes that can guide methane strategies, climate finance and waste-system planning.

3. **Uncertainty-aware result**

The deterministic result and Monte Carlo result are intentionally different: AD-bio-CNG has the largest central global benefit, while prevention is the most robust country-level winner.

4. **Readiness typology**

The readiness layer is the main novelty hook. It moves the manuscript beyond "more research is needed" and toward policy sequencing.

5. **Supplementary transparency**

The SI now includes pathway equations, parameters, Monte Carlo distributions, readiness scoring, missing-data diagnostics, city-layer validation, GWP20, landfill capture, nutrient-credit and bio-CNG stress tests.

## Main Risks

1. **ecoinvent 3.12 limitation**

The central AD-bio-CNG result still uses ecoinvent 3.1 as the biogas-yield proxy because 3.12 exchange-level access is unavailable. This is the main technical vulnerability for Nature Sustainability. The current mitigation is transparency plus sensitivity analysis. A 3.12 licence or independent literature-derived biogas-yield meta-range would strengthen this substantially.

2. **Final figures are not yet publication-grade**

Standalone draft figures exist and the final multi-panel architecture is defined, but composite Nature-style figures still need to be generated and polished.

3. **The methane model is first-pass, not full FOD**

The first-pass methane screen is acceptable for prioritization if framed carefully, but reviewers may ask for first-order decay timing or inventory comparison. GWP20 and landfill-capture sensitivities help, but full FOD remains a possible revision request.

4. **Composting is still under-quantified**

Nutrient sensitivity and soil-carbon justification are now included, but no fully spatial soil-carbon model is implemented. This is acceptable if composting is framed as conservative in the GWP100 central screen.

5. **Missing data are uneven**

The 65 missing/insufficient countries are now documented. The issue is strongest in low-income countries and Sub-Saharan Africa. This should be framed as a data-capacity and readiness finding, not hidden.

## Ready For Presubmission Inquiry?

Yes.

The presubmission inquiry can be sent with the current data. It should present the paper as an Analysis and be explicit that the work is a global screening framework rather than a facility-specific LCA. The ecoinvent limitation should be described briefly, but not allowed to dominate the pitch:

- process emissions and nutrient values come from accessible ecoinvent 3.12 documentation;
- biogas yield uses a transparent ecoinvent 3.1 cutoff proxy because exchange-level 3.12 access was unavailable;
- uncertainty and stress tests propagate the parameters most likely to affect the AD-bio-CNG ranking.

For presubmission, final figures are helpful but not mandatory if the pitch is concise and includes the core numbers.
